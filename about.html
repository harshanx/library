About The System

This web-based application allows librarians to efficiently manage book inventory,user registration issuing and returning of books,and viewing history, all from a clean dashboard.

Key Features

  • Book Issueing and Returning
  • Admin and User Login
  • Searchable Book Catalog
  • Fine Calculation for Late Returns
  • Email notification
